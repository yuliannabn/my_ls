# my_ls

Recreation of the ls command for Unix terminals.

### flags:

* -l (long listing)
* -a (list hidden)
* -d (directory name)
* -R (recursive)
* -r (sort reversed)
* -t (sort by mod. date)

# COMPILATION:
> make re

# USAGE:
The usage is the same as the original ls'.

> ./my_ls [FLAGS] [PATHS]

No path specified will be taken as "."
